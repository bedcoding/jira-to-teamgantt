사용자 목록 관리
ID 얻는 법 (개발자용)
  1. TeamGantt에서 아무 프로젝트 진입 후 상단 탭 중 People 클릭
  2. Chrome 개발자도구 (⌥⌘I) → Network 패널 열고 필터에 v1/ 입력
  3. People 페이지를 한 번 새로고침 → 아래 요청이 잡힘
  4. Response 탭의 JSON 전체 복사
  5. 아래 textarea에 붙여넣고 병합 적용
GET https://api.teamgantt.com/v1/companies/{companyId}/resources/company
응답 통째로 붙여도 id / name 필드만 추출됩니다.
